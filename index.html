// why chai-tailwind

No build tools.
Just utility classes.

Chai Tailwind brings the power of utility-first CSS to any project — no Node.js, no PostCSS, no config files. Just one script tag.

⚡

Drop a CDN script, start using classes immediately.

Animations

🖱️

Scale, tilt, color change — all with a single class.

// getting started

Up and running in 30 seconds.

index.html
<!DOCTYPE html>
<html>
<head>
  <!-- Add just this one line -->
  <script src="https://cdn.jsdelivr.net/npm/chai-tailwind@1.1.0/dist/chai-tailwind.js"></script>
</head>
<body>
  <div class="chai-bg-amber chai-text-black chai-p-4 chai-rounded-lg">
    Hello World 🚀
  </div>
</body>
</html>
terminal + JS
# Install via npm
npm install chai-tailwind

# Then import in your entry file
import "chai-tailwind";

// That's it. Styles initialize automatically.
example.html
<!-- Gradient button -->
<button class="
  chai-bg-gradient-to-r-blue-purple
  chai-text-white chai-p-4
  chai-rounded-full chai-shadow-md
  chai-hover-scale chai-transition
">Click me 🚀</button>

<!-- Shade-based card -->
<div class="chai-bg-purple-100 chai-border
  chai-border-purple-500 chai-rounded-xl chai-p-6">
  <h2 class="chai-text-purple-600 chai-font-bold">
    Card Title
  </h2>
</div>

<!-- Animated badge -->
<span class="chai-bg-amber chai-text-black
  chai-p-2 chai-rounded-full chai-animate-bounce">
  New ✨
</span>

Every class you need, prefixed with chai-

All utility classes follow a predictable naming convention. Every class is prefixed with chai- to avoid conflicts with existing stylesheets.

// installation

Ready in one command.

Install via npm or drop a script tag. Either way, you're styling in seconds.

📦 npm

For bundler-based projects (Vite, Webpack, etc.)

npm install chai-tailwind

Then: import "chai-tailwind"

For plain HTML — paste in <head>

No build step. Works in any browser.
